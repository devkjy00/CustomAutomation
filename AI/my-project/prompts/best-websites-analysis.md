# Best Websites Analysis Prompt

## Purpose
질문 유형을 분석하여 가장 좋은 답변을 제공할 수 있는 웹사이트를 추천합니다.

## Template

```
질문: "{{originalQuery}}"
검색어: "{{searchKeywords}}"

이 질문에 가장 좋은 답변을 제공할 수 있는 웹사이트 종류를 분석하세요.

질문 유형별 최적 웹사이트:
- 날씨: 기상청, weather.com, 네이버 날씨
- 뉴스/최신 정보: 언론사, 뉴스 포털, 전문 미디어
- 주식/금융: 증권사, 금융정보 사이트, 거래소
- 기술/개발: Stack Overflow, GitHub, 기술 블로그, 공식 문서
- 제품 정보: 공식 홈페이지, 리뷰 사이트, 쇼핑몰
- 일반 정보: 위키백과, 공식 사이트, 전문 정보 사이트
- 엔터테인먼트: 공식 사이트, 포털, 커뮤니티

출력 형식 (최대 3개, 구체적인 사이트명):
[사이트1, 사이트2, 사이트3]

예시:
- "오늘 서울 날씨" → [기상청, 네이버 날씨, weather.com]
- "최신 AI 뉴스" → [TechCrunch, AI 전문 미디어, IT 뉴스]
- "삼성전자 주가" → [네이버 증권, 한국거래소, 증권사]

사이트 목록만 출력하세요:
```

## Variables
- `{{originalQuery}}`: 사용자의 원본 질문
- `{{searchKeywords}}`: 생성된 검색어

## Expected Output
추천 웹사이트 배열 형식: [사이트1, 사이트2, 사이트3]
