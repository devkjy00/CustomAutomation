# Search Results Filtering Prompt

## Purpose
검색 결과에서 핵심 정보만 추출하고 불필요한 내용을 제거합니다.

## Template

```
사용자가 "{{query}}"에 대해 질문했습니다.
추천 정보 출처: {{targetWebsites}}

검색 결과:
{{searchResults}}

지침:
- 추천 정보 출처와 관련된 검색 결과에 더 높은 우선순위 부여
- 각 검색 결과의 핵심 정보를 유지하세요 (숫자, 날짜, 사실, 주요 내용)
- "방문하세요", "확인하세요" 같은 불필요한 안내 문구만 제거
- 구체적인 정보(기업명, 제품명, 통계, 날씨 데이터 등)는 반드시 보존
- 각 결과당 2-3문장으로 요약

출력 (중요도 순으로 정렬):
[1]
[2]
[3]
```

## Variables
- `{{query}}`: 사용자 질문
- `{{targetWebsites}}`: 추천 정보 출처 목록
- `{{searchResults}}`: 원본 검색 결과 (번호, 제목, 내용, 출처)

## Expected Output
번호별로 필터링된 검색 결과 요약:
```
[1] 핵심 정보 요약 (2-3문장)
[2] 핵심 정보 요약 (2-3문장)
[3] 핵심 정보 요약 (2-3문장)
```
